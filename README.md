# Rift Lab / A1 eSports

GitHub-only League of Legends dashboard for the A1 eSports friend group.

## Architecture

```text
GitHub Actions scheduled job
  -> RIOT_API_KEY from GitHub Secrets
  -> Riot API
  -> data/rift-lab.json
  -> GitHub Pages
```

The browser never calls Riot directly. It only fetches the public processed JSON file generated by GitHub Actions.


## GitHub Setup

1. Open the repository on GitHub.
2. Go to **Settings -> Secrets and variables -> Actions**.
3. Add a repository secret:

```text
RIOT_API_KEY = RGAPI-your-key-here
```

4. Go to **Settings -> Pages**.
5. Under **Build and deployment**, set **Source** to **GitHub Actions**.
6. Go to **Actions -> Update Rift Lab data and deploy**.
7. Click **Run workflow**.

After the workflow finishes, the site is served from:

```text
https://gennnnnnnnn.github.io/a1esports/
```

The workflow also runs every 6 hours.

## Local Commands

Generate sample data:

```bash
npm run update:data:sample
```

Validate scripts:

```bash
npm run check
```

Generate live data locally:

```bash
RIOT_API_KEY=RGAPI-your-key npm run update:data
```

On Windows PowerShell:

```powershell
$env:RIOT_API_KEY="RGAPI-your-key"; npm run update:data
```

## Riot Routing

The generator preserves the VN2 routing that was already debugged:

```text
ACCOUNT-V1 -> asia.api.riotgames.com
MATCH-V5   -> sea.api.riotgames.com
SUMMONER-V4 / LEAGUE-V4 -> vn2.api.riotgames.com
```

Ranked queues:

```text
420 = Ranked Solo/Duo
440 = Ranked Flex
```

## Security

Do not commit:

- Riot API keys
- `.env` files
- Google OAuth tokens
- private Google Sheet edit links

Only public processed dashboard JSON belongs in this repository.
